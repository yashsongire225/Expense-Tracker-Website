# 💰 Expense Tracker Website

A modern **Expense Tracker Management System** built using **Python, Streamlit, SQLite, Pandas, NumPy, and Matplotlib**. The application helps users record, manage, analyze, and visualize their daily expenses through an interactive web dashboard.

## 📌 Project Overview

Managing personal expenses is an important part of financial planning. This project provides a simple and user-friendly interface where users can:

* Add new expenses
* View all saved expenses
* Delete unwanted expenses
* Analyze spending patterns
* Visualize expenses using charts

The application stores data in a **SQLite database** and displays analytics using **Pandas** and **Matplotlib**.

# 🚀 Features

### ➕ Add Expense

* Enter expense title
* Enter amount
* Select category
* Select date
* Save expense into database

### 📋 View Expenses

* Display all stored expenses
* Interactive data table
* Automatically updated after new entries

### 📊 Expense Analytics

* Total Expenses
* Average Expense
* Highest Expense
* Category-wise expense analysis
* Pie Chart Visualization

### 🗑 Delete Expense

* View all expense records
* Delete any expense using its ID
* Database updates instantly

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Expense-Tracker-Website.git
```

Go inside the project

```bash
cd Expense-Tracker-Website
```

Install required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

The application will open automatically in your browser.
